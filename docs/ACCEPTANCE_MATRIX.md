## 100% 定义

| 类别 | 100% 标准 |
| --- | --- |
| 页面 | 桌面、手机、明暗主题下可访问；按钮不挤压、不假点击；loading、空状态、错误提示清楚 |
| 接口 | 前端调用有后端映射；成功和失败返回结构稳定；真实 OCI 返回不伪造；超时有明确提示 |
| 高危操作 | 必须有二次确认、操作审计、失败原因；只在专用测试资源上验收后标记完成 |
| Telegram Bot | 菜单中文正常、回调真实、权限清楚、执行结果可追踪；高危动作不静默执行 |
| 脚本 | Linux LF 换行、重复执行安全、低配 VPS 可用；安装、更新、回滚、体检都有可复现命令 |
| 文档 | README 保持简洁，详细验收记录放 docs；当前状态、未完成边界和部署命令清楚 |
